사용자 입력
    '경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)' 출력
    '시도할 회수는 몇회인가요?' 출력

결과 출력
    '최종 우승자 : ' 출력

필요한 기능
자동차 이름 생성
    입력받은 문자가 5자리 이하인 경우
    입력받은 문자가 공백이 아닌 경우
    입력받은 문자가 중복되지 않는 경우
자동차의 전진 또는 멈춤
    전진 : 0~9 사이에서 무작위 값을 구해서 4 이상일 경우
    멈춤 : 0~9 사이에서 무작위 값을 구해서 3 이하일 경우
사용자의 횟수 입력
    1 이상의 값인 경우
    입력받은 숫자가 문자가 아닌 경우
우승자 선별 기능
    가장 많이 전진한 자동차의 결과 출력
    우승자가 여러명일 경우 쉼표를 이용하여 구분

IllegalArgumentException 예외 발생
    입력받은 문자가 5자리수를 초과하는 경우
    입력받은 문자가 공백인 경우
    입력받은 문자가 중복되는 경우
    입력받은 숫자가 문자인 경우
